# 天气查询工具

一个基于高德地图API的网页版天气查询工具，支持城市天气查询、实时温度显示、未来天气预报等功能。

## 功能特性

- 🌤️ **实时天气查询**：根据城市名称查询当前天气状况
- 🌡️ **温度显示**：显示当前温度、今日最低和最高温度
- 📅 **未来预报**：显示未来两天的天气信息
- 🏞️ **动态背景**：根据天气状况自动切换背景图片
- 💡 **智能建议**：提供穿衣建议和出行提示
- 🎨 **半透明设计**：清晰显示背景图片，同时保证文字可读性
- 📱 **响应式布局**：适配不同屏幕尺寸
- 😊 **友好提示**：网络问题和城市未找到时显示友好提示

## 技术栈

- **前端框架**：HTML5 + CSS3 + JavaScript (ES6+)
- **天气API**：高德地图Web服务API
- **样式设计**：Flex布局、响应式设计、半透明效果
- **部署方式**：支持本地运行和Gitee Pages部署

## 快速开始

### 本地运行

1. 克隆或下载项目到本地
2. 在浏览器中直接打开 `index.html` 文件
3. 输入城市名称，点击查询按钮或按回车键
4. 查看天气信息

### 部署到Gitee Pages

1. 在Gitee创建一个公开仓库
2. 将项目代码推送到Gitee仓库
3. 进入仓库 → 服务 → Gitee Pages
4. 分支选择 `master`，目录选择 `根目录`
5. 点击「启动」，等待部署完成
6. 访问生成的链接即可

## 使用说明

1. **查询天气**：在输入框中输入城市名称（如：北京、上海、广州等）
2. **快捷查询**：输入城市后按回车键即可查询
3. **查看详情**：页面显示当前天气、温度、未来预报、穿衣建议和出行提示
4. **背景变化**：根据天气状况自动切换背景图片

## API配置

项目使用高德地图天气API，如需更换API密钥：

1. 访问 [高德地图开发者平台](https://lbs.amap.com/) 注册账号
2. 创建应用，获取Web服务API密钥
3. 打开 `script.js` 文件，将第3行的 `API_KEY` 替换为您的密钥

```javascript
// 修改此处的API密钥
const API_KEY = '您的高德地图API密钥';
```

## 注意事项

1. 确保项目中的图片路径正确，特别是背景图片
2. 高德地图API有免费额度限制，请勿滥用
3. 建议使用最新版浏览器以获得最佳体验
4. 如需自定义背景图片，将图片放入 `pic` 文件夹，命名规则与现有图片一致
